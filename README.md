# Projeto base para aula de Acessibilidade

## Atividade 2

### Melhorar a usabilidade para quem tem baixa visão

Criar um botão que altere o **contraste da tela**.

#### Passos para implementação

1. Criar um botão na interface.
2. Definir `estadoContraste ← falso`.
3. Quando o usuário clicar no botão:
4. Alterar `estadoContraste`.
5. Se `estadoContraste = falso`, então:
6. Ativar modo **alto contraste** que altere as cores (fundo escuro e texto claro).
7. `estadoContraste ← verdadeiro`.
8. Caso contrário, restaurar as **cores padrão da interface**.
