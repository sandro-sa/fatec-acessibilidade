# Projeto base para aula de Acessibilidade

* Adicionar um botão que aumente e dimiue o contraste da página
* Adicionar um botão que mude a cor da página para o dark e light mode.
* Criar um container de acessibilidade para adicionar esses botões para evitar que quebre o navegador.
* Posicionar a aba de acessibilidade acima do menu de navegação e fixá-lo.
* Criar o estilo do contraste, light e dark mode no CSS
* Fazer um script no JavaScript para implementar as funções.
